# Конспект лекцій: Периферійні пристрої ЕОМ

## 1. Правила електробезпеки та ESD
[cite_start]Дотримання правил безпеки запобігає виникненню електричних пожеж, травм та смертельних випадків під час роботи всередині комп'ютера[cite: 21].

### Електробезпека
* [cite_start]**Висока напруга:** Деякі компоненти, як блоки живлення та частини принтерів, містять високу напругу навіть після відключення пристрою[cite: 35, 36].
* [cite_start]**Заземлення:** Електрообладнання повинно бути заземлено для створення шляху струму з найменшим опором у разі несправності[cite: 39]. [cite_start]Зазвичай комп'ютер підключають до землі через штекер живлення[cite: 40].
* [cite_start]**Адаптери:** Заміна адаптерів змінного струму на невідповідні типи може призвести до пошкодження обладнання[cite: 38].

### Електростатичний розряд (Electrostatic Discharge, ESD)

* [cite_start]**Природа явища:** ESD виникає при накопиченні електричного заряду на поверхні, що контактує з іншою, протилежно зарядженою поверхнею[cite: 42].
* [cite_start]**Рівні напруги:** Людина відчуває розряд напругою не менше 3000 В[cite: 47]. [cite_start]Проте напруга менше 30 В може пошкодити комп'ютерні компоненти[cite: 51].
* **Засоби захисту:**
    * [cite_start]Використовуйте антистатичні наручні браслети та заземлені килимки[cite: 55].
    * [cite_start]Зберігайте компоненти в антистатичних мішках до моменту встановлення[cite: 54].
    * [cite_start]Практикуйте "само-заземлення", торкаючись заземленого обладнання[cite: 52].

## 2. Форм-фактори корпусів настільних комп'ютерів
[cite_start]Корпус забезпечує каркас для підтримки, захисту та охолодження внутрішніх компонентів[cite: 66].

| Тип корпусу | Опис та характеристики |
| :--- | :--- |
| **Горизонтальний** | Розташовувався на столі горизонтально, монітор ставився зверху. [cite_start]Популярний для HTPC[cite: 81, 82]. |
| **Повнорозмірна вежа** | Вертикальний корпус для підлоги. [cite_start]Забезпечує простір для багатьох дискових накопичувачів та плат адаптера[cite: 84, 85]. |
| **Компактна вежа** | Менша версія вежі (Mini-Tower або SFF). [cite_start]Часто зустрічається в корпоративному середовищі[cite: 87, 88]. |
| **Моноблок** | Усі компоненти інтегровані в дисплей. [cite_start]Можливості оновлення зазвичай обмежені[cite: 91, 92]. |

## 3. Блок живлення (БЖ)
[cite_start]Блок живлення перетворює змінний струм (AC) із розетки на постійний струм (DC) нижчої напруги[cite: 98, 99].

### Основні характеристики та конектори
* [cite_start]**ATX12V:** Сьогодні це найпоширеніший блок живлення на ринку[cite: 103].
* [cite_start]**EPS12V:** Спочатку розроблений для серверів, але зараз використовується в потужних настільних комп'ютерах[cite: 105].
* [cite_start]**Напруги:** 3,3 В та 5 В використовуються цифровими схемами, а 12 В — для двигунів накопичувачів та вентиляторів[cite: 135].



**Основні роз'єми:**
1. [cite_start]**20/24-контактний:** Для живлення материнської плати[cite: 112, 113].
2. [cite_start]**SATA:** Для підключення сучасних дискових накопичувачів[cite: 115, 116].
3. [cite_start]**Molex:** Для живлення жорстких дисків, оптичних приводів тощо[cite: 117, 118].
4. [cite_start]**PCIe:** 6/8-контактний роз'єм для живлення відеокарт[cite: 130, 131].

## 4. Основні компоненти материнської плати
[cite_start]Материнська плата є основою комп'ютера, що містить шини для з'єднання всіх компонентів[cite: 196, 197].



1. **Центральний процесор (CPU):** "Мозок" комп'ютера; [cite_start]обробляє інструкції та команди[cite: 206, 293].
2. [cite_start]**Оперативна пам'ять (RAM):** Тимчасове місце для зберігання даних і програм[cite: 207].
3. [cite_start]**Чіпсет (Chipset):** Керує взаємодією апаратного забезпечення[cite: 209].
    * [cite_start]**Північний міст:** Контролює доступ до RAM та відеокарти[cite: 257].
    * [cite_start]**Південний міст:** Дозволяє процесору взаємодіяти з повільними пристроями (USB, HDD)[cite: 259].
4. [cite_start]**Слоти розширення:** Роз'єми (PCIe, PCI) для додаткових плат[cite: 208].
5. [cite_start]**BIOS / UEFI:** Програмне забезпечення для завантаження системи та виконання POST[cite: 213, 214].
6. [cite_start]**SATA роз'єми:** Інтерфейс для накопичувачів із підтримкою "гарячої заміни"[cite: 228, 229].
7. [cite_start]**Внутрішній USB:** Роз'єми для підключення портів корпусу до плати[cite: 233].

### Системи охолодження

* [cite_start]**Пасивне:** Радіатори на чіпах, які не потребують енергії[cite: 312, 313].
* [cite_start]**Активне:** Вентилятори, для роботи яких потрібна електроенергія[cite: 312, 314].
