# Конспект лекцій: Периферійні пристрої ЕОМ

## 1. Правила електробезпеки при роботі з внутрішніми компонентами ПК
Дотримання правил безпеки запобігає травмам та пошкодженню обладнання.

* **Електробезпека:** Деякі компоненти (блоки живлення, монітори) зберігають високу напругу навіть після вимкнення. Обладнання має бути заземлене.
* **Електростатичний розряд (ESD):** Виникає при контакті поверхонь із різними зарядами. Людина відчуває розряд від 3000 В, а компоненти ПК псуються від напруги менше 30 В.
* **Захист:** Використання антистатичних браслетів та килимків.

![Антистатичний браслет та килимок](https://upload.wikimedia.org/wikipedia/commons/thumb/0/05/Antistatic_wrist_strap_and_mat.jpg/640px-Antistatic_wrist_strap_and_mat.jpg)

## 2. Форм-фактори корпусів настільних комп'ютерів
Корпус захищає внутрішні деталі та забезпечує їх охолодження.

| Форм-фактор | Опис |
| :--- | :--- |
| **Horizontal** | Горизонтальний, для HTPC (домашніх кінотеатрів). |
| **Full Tower** | Велика вежа для потужних систем. |
| **Compact Tower** | Міні-вежа (SFF), економить місце. |
| **All-in-one** | Моноблок (комп'ютер у моніторі). |

## 3. Блок живлення (БЖ)
Перетворює змінний струм (AC) на постійний (DC) з напругами 3.3В, 5В та 12В.

* **ATX12V:** Найпоширеніший стандарт сьогодні.
* **EPS12V:** Для серверів та потужних станцій.

### Конектори БЖ:
1. **20/24-pin:** Основне живлення плати.
2. **SATA:** Для сучасних дисків (SSD/HDD).
3. **Molex:** Для старих дисків та вентиляторів.
4. **4/8-pin CPU:** Живлення процесора.
5. **6/8-pin PCIe:** Живлення відеокарт.

## 4. Основні компоненти материнської плати
Материнська плата з'єднує всі вузли системи через шини.

1. **CPU (Процесор):** Обробляє дані ("мозок").
2. **RAM (Оперативна пам'ять):** Тимчасова пам'ять.
3. **Чіпсет:** Північний міст (швидкі шини) та Південний міст (периферія).
4. **Слоти розширення (PCIe):** Для відеокарт та інших плат.
5. **BIOS/UEFI:** Програма для завантаження системи.
6. **SATA порти:** Для підключення накопичувачів.
7. **Внутрішні USB:** Для роз'ємів на передній панелі корпусу.

## Системи охолодження
* **Пасивне:** Радіатори (без вентиляторів).
* **Активне:** Кулери (вентилятори) та системи рідинного охолодження.
