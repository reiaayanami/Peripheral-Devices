# Конспект лекцій: Периферійні пристрої ЕОМ

## 1. Правила електробезпеки та ESD
[cite_start]Дотримання правил безпеки запобігає виникненню електричних пожеж, травм та смертельних випадків під час роботи всередині комп'ютера[cite: 21].

### Електробезпека
* [cite_start]**Висока напруга**: Деякі деталі принтера та блоки живлення містять високу напругу навіть після відключення пристрою[cite: 35, 36].
* **Заземлення**: Електрообладнання повинно бути заземлено; [cite_start]зазвичай комп'ютер підключають до землі через штекер живлення[cite: 39, 40].
* [cite_start]**Адаптери**: Заміна адаптерів змінного струму на невідповідні типи може призвести до пошкодження обладнання[cite: 38].

### Електростатичний розряд (Electrostatic Discharge, ESD)
* [cite_start]**Природа явища**: ESD виникає при накопиченні електричного заряду на поверхні, що контактує з іншою, протилежно зарядженою поверхнею[cite: 42].
* [cite_start]**Рівні напруги**: Людина відчуває розряд від 3000 В[cite: 47]. [cite_start]Проте напруга менше 30 В може пошкодити комп'ютерні компоненти[cite: 51].
* **Засоби захисту**:
    * [cite_start]Використовуйте антистатичні наручні браслети та заземлені килимки[cite: 55].
    * [cite_start]Зберігайте компоненти в антистатичних мішках до встановлення[cite: 54].
    * [cite_start]Практикуйте "само-заземлення", торкаючись заземленого обладнання[cite: 52].

## 2. Форм-фактори корпусів настільних комп'ютерів
[cite_start]Корпус забезпечує каркас для підтримки, захисту та охолодження внутрішніх компонентів[cite: 66].

| Тип корпусу | Опис та характеристики |
| :--- | :--- |
| **Горизонтальний** | [cite_start]Розташовувався на столі горизонтально, монітор ставився зверху[cite: 81]. [cite_start]Часто для HTPC[cite: 82]. |
| **Повнорозмірна вежа** | [cite_start]Вертикальний корпус для підлоги, забезпечує багато місця для компонентів[cite: 84, 85]. |
| **Компактна вежа** | Менша версія вежі (Mini-tower або SFF). [cite_start]Часто в корпоративному середовищі[cite: 87, 88]. |
| **Моноблок** | [cite_start]Усі компоненти інтегровані в дисплей[cite: 91]. [cite_start]Можливості оновлення обмежені[cite: 92]. |

## 3. Блок живлення (БЖ)
[cite_start]Блок живлення перетворює змінний струм (AC) із розетки на постійний струм (DC) нижчої напруги[cite: 99].

### Основні стандарти та конектори
* **ATX12V**: Найпоширеніший сьогодні; [cite_start]має роз'єм для живлення ЦП (CPU)[cite: 103].
* [cite_start]**EPS12V**: Спочатку для серверів, зараз для потужних настільних ПК[cite: 105].
* [cite_start]**Напруги**: 3,3 В та 5 В для цифрових схем, 12 В — для двигунів дисків та вентиляторів[cite: 135].


**Основні роз'єми:**
1. [cite_start]**20/24-контактний**: Живлення материнської плати[cite: 112, 113].
2. [cite_start]**SATA**: Для підключення сучасних дискових накопичувачів[cite: 116].
3. [cite_start]**Molex**: Для жорстких дисків, оптичних приводів тощо[cite: 118].
4. [cite_start]**PCIe**: 6/8-контактний роз'єм для живлення відеокарт[cite: 130, 131].

## 4. Основні компоненти материнської плати
[cite_start]Материнська плата — це друкована плата (PCB), що містить шини для з'єднання всіх компонентів[cite: 197].


1. **Центральний процесор (CPU)**: "Мозок" комп'ютера; [cite_start]обробляє інструкції[cite: 206].
2. [cite_start]**Оперативна пам'ять (RAM)**: Тимчасове місце зберігання даних і програм[cite: 207].
3. [cite_start]**Чіпсет (Chipset)**: Керує взаємодією апаратного забезпечення[cite: 209].
    * [cite_start]**Північний міст**: Контролює доступ до RAM та відеокарти[cite: 257].
    * [cite_start]**Південний міст**: Дозволяє взаємодіяти з USB, HDD та слотами розширення[cite: 259].
4. [cite_start]**Слоти розширення**: Роз'єми (PCIe, PCI) для додаткових плат[cite: 208].
5. [cite_start]**BIOS / UEFI**: Програмне забезпечення для завантаження та POST[cite: 213, 214].
6. **SATA роз'єми**: Інтерфейс для накопичувачів; [cite_start]підтримує "гарячу заміну"[cite: 228, 229].
7. [cite_start]**Внутрішній USB**: Роз'єм для підключення портів корпусу до плати[cite: 233].

### Системи охолодження
* [cite_start]**Пасивне**: Використання радіаторів (без споживання енергії)[cite: 312, 313].
* [cite_start]**Активне**: Використання вентиляторів (потребують живлення)[cite: 312, 314].
